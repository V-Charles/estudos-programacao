# 📚 Laboratório de Estudos | Programação

Bem-vindo ao meu repositório central de estudos! 🚀

Este repositório foi criado para centralizar, organizar e versionar todos os meus projetos de estudo, exercícios práticos, laboratórios de faculdade, curso técnico e cursos extracurriculares (Udemy, etc.). 

A ideia é manter um histórico limpo e rastreável do meu aprendizado e evolução contínua em desenvolvimento de software.

Os projetos estão organizados em pastas raiz por tecnologia e, internamente, subdivididos pelo contexto do estudo (instituição, curso ou projeto específico).

```text
estudos_programacao/
│

├── html_css/         # Fundamentos de marcação, estilo e layouts
